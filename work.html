C++程式設計《實時聊天室》

使用 visual studio 2019 編寫而成，基於 Winsock 和 TCP/IP 協議達成網路聊天功能, 並使用了多執行緒來實現多人同時聊天。

C++遊戲設計 《銀行大戰》

利用 visual studio 2019 編寫而成，使用了物件導向程式設計的基礎和 STL 的概念來設計，並添 加了 EasyX 圖形庫來進行介面設計，這個專案的開發主要是用來練習 C++的編寫，例如物件導 向，指標，容器等等

Python OpenCV 《人臉辨識》

使用 VSCode 編寫而成，基於 OpenCV 電腦視覺庫，來使用數據訓練獲得人臉特徵，並結合人臉實時檢測， 實現了人臉辨識的功能。

C++程式設計《音樂播放器》

利用 visual studio 2019 編寫而成，使用了一些 STL 和物件導向的基礎概念，包括 vector，map、 迭 代器等等，使其能更有效率的讀取文件，這個專案的開發主要也是用來練習 C++的編寫以及 熟悉 STL 的概念

Python 強化學習 Q-learning《人工智慧玩迷宮》

使用 VSCode 編寫而成，基於強化學習的 Q-learning 算法實現了人工智慧玩迷宮，讓 AI 從原本 需要花費 300 多步才能抵達終點，到最後僅需 40 步就能完成。

unreal4 遊戲開發《modern swordsman》

這是使用 unreal4 遊戲引擎開發的簡易 3D ACT 遊戲,實現了戰鬥系統，包括格擋及暗殺，並添 加了 NPC 對話功能和任務系統以及簡易劇情。

Blender 動畫風格的頭部 3D 建模
